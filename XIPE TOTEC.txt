# Installation Guide

## System Requirements
- Linux x86‑64 (Ubuntu 20.04+ or WSL2)
- Python 3.10+
- RAM: 8 GB (TITAN) / 16 GB (MONSTER)
- Disk: 5 GB (TITAN) / 6 GB (MONSTER)

